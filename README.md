# scranner
Сканирует TCP порты в заданном диапозоне.

Запукс: scanner.py

Сначала вводится левая граница диапозона, затем правая, далее выводятся все открытые порты в введеном диапозоне.

Пример работы:

Set range:

First: 6000

Second: 7000

Starting scan

Port 6258: OPEN

Port 6263: OPEN

Port 6942: OPEN
